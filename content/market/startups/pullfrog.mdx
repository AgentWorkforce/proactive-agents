## Open-source details

- **Repo**: [github.com/pullfrog/pullfrog](https://github.com/pullfrog/pullfrog)
- **License**: MIT
- **Stack**: TypeScript, Node.js 24, esbuild, pnpm
- **Agent harnesses**: Claude Code (`@anthropic-ai/claude-code`) and OpenCode (`opencode-ai`)
- **MCP server**: Purpose-built with 24+ tools for git/GitHub operations via `fastmcp`
- **Runtime**: GitHub Actions (`ubuntu-latest` runners)

## Proactive agent primitives

Pullfrog implements all three proactive agent primitives:

- **Clock**: Cron-based scheduled runs for periodic automation
- **Listener**: GitHub webhook triggers on new issues, PRs, CI failures, review submissions, and @mentions
- **Inbox**: Delivers results as PR comments, reviews, issue labels, and new pull requests directly in GitHub

## Persistent memory

Two-tier learnings system:
- **Repo-level**: Durable facts about the codebase (package manager, test commands, architecture patterns) that persist across all runs
- **PR-level**: Rolling snapshot scoped to individual PRs (change objectives, prior review progress, unresolved questions)

## Competitive positioning

Positions against GitHub Copilot (pay-per-run vs per-seat, programmable triggers vs human-only, BYOK vs closed model set) and vendor agent actions like Claude Code Action and OpenAI Codex (pre-built tooling, persistent state, incremental review, eight operational modes with distinct permissions).
