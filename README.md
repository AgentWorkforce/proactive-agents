# Proactive Agents

The marketing site for [AgentWorkforce](https://github.com/AgentWorkforce)'s positioning around **proactive agents** — the runtime for agents that act on their own (schedules, triggers, watchers, durable wake/sleep).

Live: [proactiveagents.dev](https://proactiveagents.dev)

```bash
npm install
npm run dev    # http://localhost:3000
```

Built with Next.js, Tailwind, GSAP, and MDX. Deployed as a static export to Cloudflare Pages on push to `main`.
